Prototype: Design Project

The Arduino Mega microcontroller is being used in an IoT-based monitoring and automated watering system for residential plants, specifically chili pepper, calamansi, and basil, on Tuntungin Street. The system uses sensors to monitor environmental parameters like soil moisture and solenoid valve. Cultivators can remotely control the watering process through a web-based application, ensuring efficient water usage. Real-time data from sensors is displayed on a user-friendly interface, allowing for data collection. The integration of Arduino Mega allows for seamless communication between hardware components and the web-based application. This research aims to improve gardening practices, conserve water resources, and promote plant growth.

HTML | CSS | Bootstrap | JavaScript | C++ | PHP | XAMPP
Other Emebedded IoT Projects

The Internet of Things (IoT) home security system integrates technology and services to provide a secure, high-quality living environment. It monitors the entire home and triggers alerts when security is compromised. This system addresses limitations of conventional security systems, such as real-time monitoring and notifications, preventing property damage and discomfort. Sensors interconnected to the ESP8266 enable the system to respond to situations like door entry, gas leaks, and fires.
